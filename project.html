About the project

What is a MemeBot2021?



MemeBot2021 is a chatbot created for everyone who loves meme and everyone who struggles with keeping up with latest trends in meme community.



And how does it work?

MemeBot2021 uses an API of the website ‘knowyourmeme.com’, so you’re in good hands! Just write a description of a meme you want to find, and then sit back and relax, because MemeBot2021 will search through Internet databases for you and provide all the information about the meme you’re looking for!
